# Abishek_Portfolio
Abishek Portfolio(Mine) for show my skills and projects to others and get hire
This is the portfolio website of Abishek S, a passionate MERN Stack Developer. The website showcases my skills, projects, and educational background, along with a contact form for potential collaboration.

## Technologies Used

- **HTML5**
- **CSS3**
- **Tailwind CSS** (for responsive design and utility-first styling)
- **GSAP** (for animations)
- **EmailJS** (for handling contact form submissions)
- **JavaScript** (for form validation and animations)
  
## Features

- **Header**: Contains navigation links to the Projects, Education, and Contact sections.
- **Profile Section**: Displays the name and a brief introduction, with an image.
- **Projects Section**: Showcases personal and academic projects, including links to live project pages.
- **Education Section**: Displays educational qualifications with expected graduation year.
- **Contact Section**: Contains a functional contact form, using EmailJS for sending messages.
- **Responsive Design**: The website is fully responsive, adjusting to all screen sizes using Tailwind CSS.
